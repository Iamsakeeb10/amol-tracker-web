## Design Decisions

1. **Custom SVGs for Islamic icons** — lucide-react lacks mosque/praying hands/prayer beads. Custom SVGs keep the dependency footprint minimal while providing beautiful, consistent icons matching lucide's stroke-based style.

2. **`as const` compatibility** — The `content` object uses `as const`. Changing `icon` from string to component reference works because `typeof ClipboardList` is a valid React component type. No need to remove `as const`.

3. **`t()` function limitation** — `t()` returns strings, so emoji embedded in translated text strings (badge, tagline) must be separated: remove emoji from strings, render icon components at the call site.

4. **Sizing** — Use lucide's `size` prop matching original `fontSize`, except Features cards where 24px fits the 48x48 container better than 32px.

5. **Gold coloring** — All icons use `color="var(--gold)"` for consistency with the existing design system.

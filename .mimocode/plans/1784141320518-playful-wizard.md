2. Import `Image` from `next/image`, `content` from `@/lib/content`, `useLang` from `@/context/LanguageContext`

Section layout:
- `id="screenshots"`, background `rgba(0,0,0,0.2)`, padding 96px desktop / 64px mobile
- Centered heading + subheading from `content.screenshots` via `t()`
- Margin below heading: 64px

Grid container:
- Desktop (≥1024px): 4-column grid, gap 24px
- Tablet (768–1023px): 2-column grid
- Mobile (<768px): horizontal scroll (`overflow-x: auto`, `display: flex`, `gap: 16px`, `scroll-snap-type: x mandatory`)

Each screenshot card:
- Fixed width 220px (enables horizontal scroll on mobile)
- Phone frame wrapper: border-radius 28px, border 1.5px solid rgba(201,168,76,0.3), background rgba(13,61,46,0.8), aspect-ratio 9/19, overflow hidden, box-shadow 0 20px 60px rgba(0,0,0,0.4)
- Inside: try `next/image` with `fill` layout, `objectFit: 'cover'`, `sizes="220px"`, `priority` for first image
- Fallback on error: solid emerald background with caption text centered
- Caption below phone: 13px, color rgba(255,255,255,0.5), centered, margin-top 12px

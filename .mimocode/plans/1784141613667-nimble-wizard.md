# CHUNK 6 Implementation Plan — About + Download Sections

## Context

The Amol Tracker landing page has placeholder components for About and Download. CHUNK 6 replaces these with fully styled, bilingual components matching the existing design system.

**Existing patterns observed:**
- `'use client'` directive at top
- Inline styles on elements, responsive CSS via `<style>` blocks
- `useLang()` hook with `t()` function for translations
- `content` object from `@/lib/content` for all text data
- `.section-container` class for 1200px max-width
- CSS variables for colors (e.g., `var(--emerald-deep)`, `var(--gold)`)
- Hover effects via `onMouseEnter`/`onMouseLeave` handlers

## Implementation Approach

Follow the exact pattern from `Features.tsx` and `Hero.tsx`:
1. `'use client'` at top
2. Import `content` and `useLang`
3. Inline styles on elements
4. `<style>` block at bottom for responsive CSS classes
5. No external CSS files, no Tailwind utility classes (matching existing codebase)
